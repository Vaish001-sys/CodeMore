# Week 3 - Task 3: Object Oriented Programming in Python

This is a beginner Python project made as part of my internship tasks.
It covers the basics of OOP — classes, objects, inheritance, and polymorphism.

---

## What's Inside

| File | What it does |
|------|--------------|
| `src/student.py` | Creates a Student class and shows student info |
| `src/animals.py` | Shows inheritance and polymorphism using Animal, Dog, and Cat |

---

## How to Run

Make sure Python is installed on your computer.
Open a terminal in the project folder and run:

**For student.py:**
```
python src/student.py
```

**For animals.py:**
```
python src/animals.py
```

That's it. No extra libraries needed.
