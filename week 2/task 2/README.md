# Week 2 – Task 2: Python Basics

This is a beginner Python task. It has two small scripts that show how variables and user input work in Python.

---

## What's Inside

| File | What it does |
|---|---|
| `src/hello_world.py` | Creates variables of different types and prints them |
| `src/user_input.py` | Asks the user some questions and prints a fun result |

---

## How to Run

Make sure Python is installed on your computer.
Open a terminal in this folder and run:

**Script 1:**
```
python src/hello_world.py
```

**Script 2:**
```
python src/user_input.py
```

That's it. No libraries to install. No setup needed.
