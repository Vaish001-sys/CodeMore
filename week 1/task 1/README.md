# Conditional Statements & Loops in Python

This is my Week 1 internship task. I made two Python programs that show how to use if-else conditions and loops.

---

## Files

- `src/number_checks.py` — checks if a number is even/odd and positive/negative/zero
- `src/loop_operations.py` — prints multiplication table, sum of numbers, and star pattern

---

## How to Run

**Program 1:**
```
python src/number_checks.py
```

**Program 2:**
```
python src/loop_operations.py
```

---

## What Each Program Does

### number_checks.py
- You enter a number
- It tells you if the number is even or odd
- It tells you if the number is positive, negative, or zero

### loop_operations.py
- You enter a number and it prints the multiplication table
- You enter N and it calculates the sum of first N natural numbers
- You enter rows and it prints a star pattern
